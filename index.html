NEW LEGACY ROLEPLAY is a community-driven FiveM server that focuses on providing a realistic and immersive roleplaying experience. Our server is built with custom scripts and resources that enhance gameplay while maintaining optimal performance.

We pride ourselves on creating a welcoming environment for all players, whether you're new to roleplaying or a seasoned veteran. Our team is constantly working to improve the server and add new features based on community feedback.

SERVER RULES

To ensure a fair and enjoyable experience for everyone, please follow these rules.

1

Respect All Players

Treat all players with respect. Harassment, discrimination, and bullying will not be tolerated.

2

Stay In Character

Maintain roleplay at all times. Breaking character disrupts immersion for other players.

3

No Cheating

Use of mods, hacks, exploits or any unauthorized tools will result in an immediate ban.

4

Value Life

Act as you would in real life. Avoid unrealistic actions that would put your character in danger.

5

No RDM/VDM

Random Death Match and Vehicle Death Match are strictly prohibited.

6

Follow Staff Instructions

Staff decisions are final. If you have an issue, report it through proper channels.
